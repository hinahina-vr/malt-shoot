# ショット・オブ・モルト 世界観設計書 v8.0
## 「汁の反乱 〜冬の時代の遺産〜」

> 詳細なストーリーは [story_concept.md](./story_concept.md) を参照

---

## 🌍 世界観

かつての「ウイスキー冬の時代」（1980〜2000年代）が、
最近のウイスキー人気に嫉妬し、再び冬の時代をもたらそうとしている。

暴走するウイスキー（汁）を止めるため、
甲斐駒ヶ岳の妖精「かい」が各地の蒸留所を巡る。

---

## 🎭 主要キャラクター

### 自機: **かい** (Kai)
| 項目 | 設定 |
|------|------|
| 正体 | 甲斐駒ヶ岳の妖精 |
| 外見 | 山伏装束×登山服。眼鏡 |
| 備考 | 能書きの王。主人公 |

### 1面ボス→仲間: **ひなひな** (Hinahina)
| 項目 | 設定 |
|------|------|
| 舞台 | 白州蒸留所 |
| 外見 | モルトバトルちゃんねるの看板娘 |
| 備考 | 白州好き。かいの山の麓に住み着いてる。1面で戦った後、仲間になる |
