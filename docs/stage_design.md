## Stage 1: 白州蒸留所（山梨）

### 基本情報
| 項目 | 内容 |
|------|------|
| ボス | ひなひな |
| BGM雰囲気 | 軽快・森のざわめき |
| 難易度 | ★☆☆☆☆ |
| 特殊ギミック | なし（チュートリアル的） |

### 背景演出
- **道中前半**: 白州の森。木漏れ日が差し込む。鹿や鳥のシルエット
- **道中後半**: 蒸留所に近づく。ポットスチルの影。琥珀色の光
- **ボス戦**: 蒸留所の熟成庫。樽が並ぶ。汁があふれている

### 道中構成
| セクション | 敵タイプ | 弾幕テーマ |
|-----------|---------|-----------|
| 森林エリア | 木の葉型・しずく型 | 散らばる緑弾 |
| 渓流エリア | 水玉型・魚型 | 流れる青弾 |
| 蒸留所入口 | 樽型・ボトル型 | 琥珀色の回転弾 |

### 中ボス
- **なし**（1面のためシンプルに）

### ボス戦（ひなひな）
| Phase | Batch | 弾幕イメージ | 難易度 |
|-------|-------|-------------|--------|
| 1 | Batch 1「北杜の名水伝説」 | 水滴が放射状に広がる | ★ |
| 2 | Batch 2「白州の森でやってんね」 | 木の葉が舞い散る | ★ |
| 3 | Batch 3「暴れ泡の木桶発酵」 | 泡立つ弾が不規則に | ★★ |
| 4 | Batch 4「消えた東蒸留所」 | 幻影弾（消えて再出現）| ★★ |
| 5 | Batch 5「フィールドバレエの夜」 | 優雅に回転する弾群 | ★★ |
| 6 | Batch 6「最善を尽くせ、しかもエーヤオであれ」 | 全方位＋誘導弾 | ★★★ |

|------|------|
| ボス | 本坊 霞 |
| BGM雰囲気 | ふわふわ・雲の上 |
| 難易度 | ★★☆☆☆ |
| 特殊ギミック | 視界不良（霧エフェクト）|

### 背景演出
- **道中前半**: 山道を登る。霧が立ち込める

- **ボス戦**: 蒸留所。空っぽの樽が転がっている（売っちゃった痕跡）

### 道中構成
| セクション | 敵タイプ | 弾幕テーマ |
|-----------|---------|-----------|
| 山麓エリア | 霧型・雪型 | 白い拡散弾 |
| 雲海エリア | 雲型・風型 | ゆっくり漂う弾 |
| 蒸留所前 | 樽型（空）・ボトル型 | 不規則な軌道 |

### 中ボス
- **からっぽの樽の群れ**（売っちゃった樽たちの亡霊）

### ボス戦（本坊 霞）
| Phase | Batch | 弾幕イメージ | 難易度 |

## Final Stage: 極北の熟成庫

### 基本情報
| 項目 | 内容 |
|------|------|
| ボス | 軽井沢 める |
| BGM雰囲気 | 静寂・悲哀・氷結 |
| 難易度 | ★★★★★★ |
| 特殊ギミック | 凍結（スロー効果）|

### 背景演出
- **道中前半**: 閉鎖された蒸留所の廃墟。時が止まっている
- **道中後半**: 凍てつく熟成庫。氷に閉ざされた樽

### 道中構成
| セクション | 敵タイプ | 弾幕テーマ |
|-----------|---------|-----------|
| 廃墟エリア | 亡霊型・残骸型 | 消えては戻る弾 |
| 氷結エリア | 氷型・雪型 | スロー効果弾 |
| 極北の入口 | 冬型・嫉妬型 | 青白い高速弾 |

### 中ボス
- **冬の時代の残滓**（過去の閉鎖蒸留所たちの集合体）

### ボス戦（軽井沢 める）
| Phase | Batch | 弾幕イメージ | 難易度 |
|-------|-------|-------------|--------|
